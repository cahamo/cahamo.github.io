Experiment #2: Tin Foil Sensor

** This exeriment is a work in progress **

Some examples of metal based sensors were found online. Noticing how much more responsive the results from the CapacitiveSensor library were when the paper clip was accidentally touched suggested that metal sensors would be better. So the obvious next iteration in the design of the instrument's keys was to try a metalic sensor.

Sesnor

For this sensor an approximately 4.7×7.5cm rectangle of cheap kitchen foil was used. This was attached, along it's shorter side, to a slightly larger piece of card using copper tape. The copper tape has conductive adhesive. One pin of a piece of breadboard jumper cable was soldered to the copper tape. The remaining edges of the tin foil were attached to the card using masking tape.

Circuit

The circuit used in experiment 1 was adapted slightly for this sensor. The only change was that a 1kΩ resistor was placed in series with the sensor. This was done to minimise the risk to the micro-processor from any static discharge from a finger through the tin foil.

Pin 2 of the Uno is configured as an output pin and is connected, via a 1MΩ resistor to input pin 3 and the 1kΩ resistor protecting the sensor.

Code

The code used is the same as that used in experiment 1: SingleSensorRawDataLogger. This code takes repeated sensor readings using the CapacitiveSensor library and reports the results over the Uno's serial port.

Methodology

The code was again run multiple times, with different sample rates of 1, 10, 20, 50, 100, 150, 200 and 255. The following tests were attempted:

  • A finger was tapped on the sensor at a steady rate.
  • A finger was tapped more rapidly.
  • A finger was moved slowly towards the sensor before finally touching it.
  • Different fingers were used to tap the sensor.

The output was displayed on the Arduino IDE's serial plotter and screenshots were taken of the results.
